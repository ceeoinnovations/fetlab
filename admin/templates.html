Templates


Links Templates Instructions

We use a data-driven website building system to build template-based websites. It allows you to build a website and update content without coding. You can maintain all the content with Google spreadsheets.

There are currently two types of templates you can use.

1. Lab Website

    • It is designed for:

    • Displaying projects and people information of a research group

    It uses:

    • Google Drive (spreadsheets)
    • GitHub

    It contains:

    1. Home page
      • Introduction
      • Partner websites
      • Projects
        • Filters
      • People
        • Filters
      • Footer
    2. Project detail page
      • List of photos
      • Description

    Code and instructions:

    https://github.com/ceeoinnovations/fetlab

    2. Product Website

    It is designed for:

    • Collecting project content (e.g., description and files such as photos, code) for a specific product from multiple people and displaying them in real-time. Originally designed for hackathon organizers

    It uses:

    • Google Drive (form, spreadsheets)
    • GitHub

    It contains:

    1. Home page
      • Filters
      • Description for a filter selected
      • Buttons for additional links
      • Submit button that links to a Google form
      • Projects
    2. Project detail page
      • Photo image gallery
      • Description
      • You might also like

    Code and instructions:

    https://github.com/ceeoinnovations/product-template
